#### 1.常量定义

​	全局变量，存放到内存中。
​	全局常量，直接解析成常数。

```java
ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.add(3);
System.out.println(list.stream().map(Integer::floatValue).collect(Collectors.toList()));
// [1.0, 2.0, 3.0]
```

* 常量是否需要分配内存？
  需要，局部变量可能覆盖常量标识符。因此不能将所有常量标识符替换。
